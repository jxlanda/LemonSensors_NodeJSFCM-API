API Endpoint

https://nodejs-fcm.herokuapp.com

La API NodeJs-FCM proporciona acceso programático para enviar mensajes a traves del servicio gratuito de Firebase Cloud Messaging.

Para informacion detallada consultar el API de Firebase Cloud Messaging siguiendo el enlace: Messages NodeJS
